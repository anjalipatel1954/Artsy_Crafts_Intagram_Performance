Instagram Analytics Dashboard for Artsy Crafts
Project Overview

This project is a Power BI dashboard created to track and analyze the performance of the Instagram resin art business page Artsy Crafts.

The dashboard helps monitor:

Post performance
Engagement metrics
Audience growth
Content strategy
Product order trends
Best posting times

The goal is to improve Instagram growth and increase product sales through data-driven insights.

Tools Used
Microsoft Power BI
Microsoft Excel
DAX Measures
Data Visualization
Dataset Fields

The dataset contains the following columns:

Field Name	Description
date	Post upload date
post_id	Unique post identifier
content_type	Reel, carousel, image, etc.
product	Resin product featured
theme	Content theme/style
posting_time	Time of posting
views	Total views
likes	Total likes
saves	Total saves
comments	Total comments
shares	Total shares
followers_gained	Followers gained from post
orders_receive	Orders received
note	Additional observations
Dashboard Pages
1. Overview Dashboard

Tracks overall Instagram performance using KPI cards and trend charts.

2. Content Performance

Analyzes which content types and themes generate the highest engagement.

3. Orders & Conversion

Measures how Instagram content converts into customer orders.

4. Posting Strategy

Identifies the best posting times and days for maximum reach and engagement.
